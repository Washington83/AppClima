# 🌥️ Previsão Do Tempo

🧪 Projeto que Consome dados através da Api Clima do Tempo, nesse projeto aprendi a consumir api e mecher com tramento de erros.

🛠️ Tecnologias Utilizadas

`Front-end: HTML, CSS, JavaScript ` <br>
`APIs: OpenWeatherMap (dados metorológicos)`

🔧 Como Rodar o Projeto: <br> 
<ul>
  <li> Faça o clone do projeto e pegue sua "API KEY", e altere no codigo JS</li>
  <li><a href="https://openweathermap.org/current#name">Chave de API Disponivel No OpenWeatherMap - Gratuita</a></li>
</ul>
